# 🎫 Prestige Planner — Event Management System

Chào mừng bạn đến với dự án **EventHub**.

## 📂 Cấu trúc dự án

```
Event-Management-Website/ (Root)
├── Event-Management-Website/  # ⚛️ Frontend (React SPA)
├── Event-Management-Backend/  # ☕ Backend (Java Spring Boot 3)
└── docs/                      # 📑 Tài liệu thiết kế
```

## 🚀 Hướng dẫn khởi chạy nhanh

### 1. Chạy Backend
```bash
cd Event-Management-Backend
docker compose up -d
./mvnw spring-boot:run
```

### 2. Chạy Frontend
```bash
cd Event-Management-Website
npm install
npm run dev
```

## 📑 Tài liệu tham khảo
Chi tiết tại: `Event-Management-Backend/docs/`
